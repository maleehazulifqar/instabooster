# InstaBooster Tool

InstaBooster Tool is a Flask-based Instagram growth dashboard developed by **FAM Developers**.

It allows users to sign up, log in, select Instagram services, place free or paid orders, verify payments through JazzCash/EasyPaisa, and track order status from a responsive dashboard.

## Features

- User signup and login
- Landing page
- Mobile-friendly dashboard
- Instagram Views, Likes, and Followers services
- Free Instagram views feature
- Paid likes and followers
- Quantity selection buttons
- Live amount calculation
- JazzCash and EasyPaisa payment options
- Payment verification
- Order status tracking
- Account settings page

## How It Works

1. User opens the landing page.
2. User creates an account or logs in.
3. User selects Instagram service.
4. User selects quantity.
5. System shows the amount to pay.
6. Free views orders are placed directly.
7. Paid orders use JazzCash or EasyPaisa.
8. Payment is checked automatically.
9. Order is placed after payment confirmation.
10. User tracks order from My Orders page.
